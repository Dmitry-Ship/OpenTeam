# OpenTeam

## Installation

```
python -m venv .venv  
```

```
source .venv/bin/activate
```

```
pip install -r requirements.txt
```

## Setup Environment variables

```
cp .env.example .env
```


## Run

```
python -m analyst.main
```

```
python -m coder.main
```

```
python -m elements_creator.main
```

```
streamlit run youtube_ui.py
```
